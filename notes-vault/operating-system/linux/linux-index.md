Ctrl + K (Cut Current to End of Line)  
Ctrl + U (Cut Current to Start of Line)

Ctrl + D (Exit Session)

Ctrl + A (Move to Start of Line)  
Ctrl + E (Move to End of Line)

Ctrl + R (Reverse Search)  
Ctrl + G (Exit Reverse Search)
