### Ctrl Shortcuts

`Ctrl + Arrow`: Jumps to the last cell in the direction of  the arrow.  
`Ctrl + Shift + Arrow`: Selects to the last cell in the data range.  
`Ctrl + Home/End`: Jumps to Home (top-left) and End (bottom-right) of range.  
`Ctrl + .`: Jumps to each corner within a selected range.  
`Ctrl + PgUp/PgDn`: Switches worksheet tabs.  

`Ctrl + +`: Add a new row (above)/column (left)  
`Ctrl + -`: Delete a row/ column

###  Function Shortcuts

`F1`: Help Pane.  
`F2`: Edit Mode for active cell. Highlights referenced cells.  
`F4`: Repeat last action. In formula bar toggle absolute/relative reference.  
`F9`: Recalculate formulas (Manual Mode). Evaluate selected portion of formula.

### Alt Shortcuts

`Alt - H - V - V`: Paste Special as Values.  
`Alt - A - T`: Add/Remove filters.  
`Alt - M - V`: Evaluate formula tool.  
`Alt - H - O - I`: Autofit column width.
