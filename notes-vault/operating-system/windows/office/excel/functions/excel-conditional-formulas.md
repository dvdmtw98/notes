### IF Statement
IF statement can be used with `AND()` and `OR()` statements.

![[excel-if-statement.png|640]]

AND and OR functions can be used in any function that expected a condition the evaluates to TRUE/FALSE.  
Negation can be performed using `NOT()` or `<>`.
