Active Directory is an **identity** management system.  
AD is a role that can be installed/enabled on Windows Servers.  
The server on which AD is installed is called **Domain Controller (DC)**.  
AD database is stored at `C:\Windows\NTDS\ntds.dit`.  

AD is an database that stores **objects** (usernames, groups, computers, etc.)  
Objects in AD are arranged into folders called **Organizational Units (OU)**.
