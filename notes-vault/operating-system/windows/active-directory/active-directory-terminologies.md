#### Domain
An Active Directory domain is a logical grouping of AD objects that share the same database, security policies, and authentication system.  
A domain is a security and administrative boundary.  

#### AD Object
An object is the basic element of Active Directory.  
Users, Groups, Printers, Computers, Shared Folders are all AD Objects.  
Objects in AD are defined/described using its attributes.  
The blueprint that decides the structure of the object is called schema.  

#### Group Policy Object (GPO)
A virtual collection of policy settings  
Controls the access permissions of AD Objects

#### Organization Unit (OU)
A subdivision within an AD into which we can place objects (users, groups, computers and other OUs).  

#### Directory Service
Provides the methods for storing directory data and making this data available to network administrators.  
Directory services run on the Domain Controller (DC).
