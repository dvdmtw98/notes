It is a feature of the Windows OS.  
If DHCP not found Windows can auto-assign IP address and subnet mask.  
It assigns a Class B IP address.  

IP Address Range: 169.254.0.1 - 169.254.255.254  
Subnet Mask: 255.255.0.0
