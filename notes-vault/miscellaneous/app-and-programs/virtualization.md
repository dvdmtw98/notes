### Virtual Box

#### Mounting Shared Folder

[Shared Folders - Oracle](https://docs.oracle.com/en/virtualization/virtualbox/6.0/user/sharedfolders.html)

Windows

```cmd
net use x: \\vboxsvr\sharename
```

Linux

```bash
mount -t vboxsf sharename mountpoint
```
