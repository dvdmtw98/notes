$x^{n-1} \equiv 1 \mod n$

n : Prime no.  
x : +ve integer not divisible by n ($x \not\equiv \mod n$)

We get this from the fact that Φ(n) = n -1 when n is prime

$x^{Φ(n)+1} \equiv x \mod n$ when GCD(x, n) = 1  
$x^n \equiv x \mod n$
