<u>Telnet (Telecommunication Network)</u>  
Uses [TCP (Transmission Control Protocol)](../4%20-%20Transport%20Layer%20Protocols/TCP%20%28Transmission%20Control%20Protocol%29.md)

Uses [TCP (Transmission Control Protocol)](../4%20-%20Transport%20Layer%20Protocols/TCP%20%28Transmission%20Control%20Protocol%29.md)
