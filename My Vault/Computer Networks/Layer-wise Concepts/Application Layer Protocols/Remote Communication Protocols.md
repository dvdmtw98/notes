<u>Telnet (Telecommunication Network)</u>  
Uses [TCP (Transmission Control Protocol)](../Transport%20Layer%20Concepts/TCP%20(Transmission%20Control%20Protocol).md)

Uses [TCP (Transmission Control Protocol)](../Transport%20Layer%20Concepts/TCP%20(Transmission%20Control%20Protocol).md)  
Allows to login and manage remove devices (Encrypted)
