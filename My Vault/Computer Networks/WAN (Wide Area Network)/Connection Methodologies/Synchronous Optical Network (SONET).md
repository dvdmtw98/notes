---
title: Synchronous Optical Network (SONET)
tags: [networking, wan]
---

Designed to carry large volumes of data over long distance on fiber optic cabling  
Supports speeds of in Gbps range

| Level  | Speed      |
| ------ | ---------- |
| OC-1   | 51.84Mbps  |
| OC-3   | 155.52mbps |
| OC-12  | 622.08Mbps |
| OC-24  | 1.244Gbps  |
| OC-48  | 2.488Gbps  |
| OC-192 | 9.953Gbps  |
