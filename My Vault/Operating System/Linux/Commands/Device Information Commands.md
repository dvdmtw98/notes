### Disk Usage

````shell
du -hc ./Download/* # Space occupied by file
df -hT # File System Usage
````

**<u>Flags</u>**:  
-s : Summarize  
-a : All files (Not just directories)  
-h : Human Readable  
-c : Total Size  
-T : Show File System Type

---

### CPU & Memory Usage

````shell
free -h # Memory Usage
lscpu # Returns CPU information
````
