---
title: xfreerdp Command
---

````shell
xfreerdp <options><username><hostname>
xfreerdp /u:<hostname> /p:<password> /v:<hostname>
````

Use "Ctrl + Alt + Enter" to exit Fullscreen

**<u>Flags</u>**:  
/port:port-no  
/size:`<width>x<height>`  
/f  
+clipboard
